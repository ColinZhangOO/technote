# Style Guide

https://www.python.org/dev/peps/pep-0008/

补充：

- 每行最大长度限制为 120 个字符。
- 私有成员要加下划线前缀，private 和 protected 均加一个。
- 尽量只 `import` module 或 class，避免频繁 `import` function（写起来太费劲），按需使用 `from`。
